Overview
GramaSethu is an Android application designed to provide real-time information about public infrastructure such as bridges and culverts. The application helps users monitor infrastructure conditions and report issues to improve safety and transparency.

Objective
The main objective of this project is to:

Provide easy access to infrastructure data
Improve public safety awareness
Enable reporting of damaged or unsafe structures
Features
View infrastructure locations on map
Real-time status updates
Report issues with images
Alerts for unsafe conditions
Simple and user-friendly interface
Technologies Used
Kotlin
Jetpack Compose
Android Studio
Google Maps API
